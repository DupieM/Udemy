## Course Content
### Section 2: Introduction to C# and .NET Framework
Set up the Visual Studio IDE, configured ReSharper, and built my first C# console application. Learned .NET architectural fundamentals including the Common Language Runtime (CLR), compilation process, and the core differences between C# and the .NET platform.

Main Folder: `HelloWorld2`
Sub Folder: `HelloWorld2 (First Application)`

### Section 3: Primitive Types and Expressions
Worked with variables, constants, type casting (implicit and explicit), scope, and operators (arithmetic, logical, and comparison). Mastered how data is stored in memory and learned how to identify and prevent data overflow issues.

Main Folder: `HelloWorld2`
Sub Folders: `Variables`, `TypeConversions`, `Operators`

### Section 4: Non-Primitive Types
Explored classes, structs, arrays, strings, and enums. Mastered the critical architectural differences in memory allocation between value types (stored on the Stack) and reference types (stored on the Heap), which is vital for performance optimization.

Main Folder: `CSharpFundamental`
Sub Folders: `CSharpFundamental (Classes)`, `ArraysPartOne`, `Strings`, `Enum`, `ReferenceTypesAndValueTypes`, `ReferenceAndValueTypesPartTwo`

### Section 5: Control Flow
Implemented conditional statements (if/else, switch/case) and iteration structures (for, foreach, while loops) to control application logic. Practiced writing interactive programs and utilized the Random class for dynamic data generation.

Main Folder: `CSharpFundamental`
Sub Folder: `Conditionals`

Main Folder: `CSharpFundamental2`
Sub Folder: `CSharpFundamental2 (Loops)`, `RandomClass`

### Section 6: Arrays and List
Learned how to store, manage, and manipulate collections of data. Highlighted the differences in use cases and performance between fixed-size arrays and dynamic, resizable lists (List<T>), utilizing built-in methods like Add(), Remove(), and Sort().

Main Folder: `CSharpFundamental2`
Sub Folders: `Arrays`, `Lists`

### Section 7: Working with Dates
Managed temporal data in C# by learning to format, manipulate, and calculate intervals using DateTime and TimeSpan structures, including parsing dates from strings and performing calendar arithmetic.

Main Folder: `CSharpFundamental2`
Sub Folders: `DatesandTime`, `CalculatingTime`

### Section 8: Workinbg with Text
Developed routines to clean, search, summarize, and format strings using standard system methods. Leveraged the mutable StringBuilder class to write memory-efficient string manipulation routines for heavy-duty text processing.

Main Folder: `CSharpFundamental3`
Sub Folders: `CSharpFundamental3 (Strings)`, `CSharpFundamental3.1 (Summarising Text)`, `CSharpFundamental3_2 (String Builder)`

### Section 9: Working with Files
Interacted with the local file system using the System.IO namespace. Practiced managing directories, processing file paths, and reading/writing file contents securely using classes like File, FileInfo, Directory, and Path.

Main Folder: `CSharpFundamental3`
Sub Folders: `CSharpFundamental3_3 (File/FileInfo)`, `CSharpFundamental3.4(Directory/DirectoryInfo)`, `CSharpFundamental3_5 (Path)`

### Section 10: Debugging Applications
Learned professional debugging techniques in Visual Studio. Used the Call Stack, Autos, and Locals windows to inspect application state step-by-step, eliminate code side effects, and write defensive code to handle potential runtime failures.

Main Folder: `CSharpFundamentalFinal`
Sub Folder: `CSharpFundamentalFinal (Debigging/Side Effects/Defensive Programming/Windows Functions)`
